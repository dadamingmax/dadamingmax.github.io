买了SuperGrok才发现：这200美元免费额度，没人告诉你怎么用

Jun 17 2026 SuperGrok>额度>Grok>Grok Build 8 minutes read (About 1143 words)

SuperGrok 订阅包含约 $200/月的 xAI API 额度，可访问 grok-imagine-image-quality、grok-imagine-video-1.5-preview、grok-4 等高级模型。但绝大多数用户只用了网页聊天，这部分额度几乎全部浪费。

本文提供两条路径：Grok Build 自动化配置，以及手动逐步操作。

核心原理

xAI 网页端走受限通道，图像和视频配额少且功能受限。通过开源工具 CLIProxyAPI（cliproxy） 将 SuperGrok 登录态转换为本地 OpenAI 兼容 API（http://127.0.0.1:8317/v1），即可在 ComfyUI、Open WebUI 等工具中直接调用订阅内的真实模型额度。

解锁后可访问的模型：

模型 能力
grok-imagine-image-quality 高质量/Pro 图像生成
grok-imagine-video-1.5-preview 更长、更连贯的视频，支持参考图/视频
grok-4 更强的对话与推理

路径一：Grok Build 自动配置（推荐）

Grok Build 是 xAI 官方 CLI coding agent，SuperGrok 用户免费使用，可直接操控本机执行配置。

安装

安装后在任意目录运行 grok，首次启动会打开浏览器完成 OAuth 登录。

路径二：手动配置

1. 安装 CLIProxyAPI

Windows

  1. 前往 Releases 页面，下载 CLIProxyAPI_windows_amd64.zip

  1. 解压至 C:\cliproxy
  2. 复制 config.example.yaml 到 config.yaml

WSL / Linux（支持 systemd 自启）

2. 登录 SuperGrok 账号

cliproxy 启动后，浏览器打开 http://localhost:8317/management.html。

  • management key 建议设置为简短字符串（如 local）
  • 点击 xAI 登录区域完成 OAuth 授权
  • Token 保存后服务即可持续调用真实 API

SuperGrok 官方订阅升级服务:cnmGrok.com。

3. 安装 ComfyUI

从 ComfyUI Releases 下载对应版本的便携包（NVIDIA / AMD / CPU），解压后运行对应的 .bat 文件，浏览器访问 http://127.0.0.1:8188。

4. 将 ComfyUI Grok 节点指向本地代理

ComfyUI 默认 Grok 节点走 Comfy 云通道。需将节点 API 端点替换为 http://127.0.0.1:8317/v1，API key 填入 management key（local）。可在节点配置中直接修改，或使用 Grok Build 指令自动完成。

5. （可选）Open WebUI 聊天界面

在 Open WebUI 的 Settings，选择 Connections，点击OpenAI 中新增连接：

即可在类 ChatGPT 界面使用 grok-4，支持文件上传、持久记忆、自定义角色。

常见问题

额度如何消耗？ 消耗的是 SuperGrok 订阅本身包含的 API 额度，非额外赠送。专业工作流单次消耗高于网页聊天，但产出效率和质量同步提升。

OAuth 登录失败？ 检查网络是否能正常访问 x.ai，必要时确保代理支持 OAuth 回调。

后续模型更新如何跟进？ 在 Grok Build 中输入”更新我的 Grok 节点和模型列表”，自动维护。

适合人群

适合：图像/视频重度创作者、ComfyUI 进阶用户、已订阅 SuperGrok 但觉得网页配额不够的用户。

不适合：偶尔生成几张图的轻度用户——网页版已经足够，无需折腾本地环境。

配置完成后，你本地运行的就是一个完整的 xAI API 工作站，而不是受限的网页聊天窗口。这才是 SuperGrok 订阅应有的使用方式。
